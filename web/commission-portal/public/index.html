[ 사용방법 ] 1. 행정심판청구서, 집행정지신청서 및 별지 제1호~제4호 본문에서 호증 표기(링크)를 더블클릭하면 오른쪽에 갑호증 또는 법령정보 요지가 열립니다. 2. 화면에 출력된 PDF·이미지·동영상 등을 다시 더블클릭하면 전체 화면으로 볼 수 있고, 더블클릭 또는 Esc로 닫을 수 있습니다. 3. 다중 PDF는 1쪽 썸네일만 보이며, 전체는 전체 화면에서 봅니다(원고크기, 좌우회전 가능). 4. 동영상은 자동 재생하지 않고, 전체 화면에서 자동 재생됩니다.
