# 저장소 경로 전수조사 (younsu)

프로젝트 루트 기준. **표기가 두 가지인 경우**는 아래 「이중 표기」를 먼저 본다.

## 1. 제출 서면(포털 `tabSources`) — 단일 정본

| 역할 | `portal-data.json`·`/serve/` 상대 경로 | 디스크(기본) | 디스크(대체) |
|------|----------------------------------------|--------------|--------------|
| 탭용 MD | `행정심판청구(최종)/260404/…` 및 직하 `260404_01_…` 등 | **`행정심판청구(최종)/`** | 같은 꼬리 경로가 **`행정심판최종본/`** 에만 있는 클론 |

- **`web/commission-portal/start.js`** — `행정심판청구(최종)/…` 를 먼저 연 다음, 파일이 없으면 **`행정심판최종본/…`** 동일 꼬리로 시도(옛 클론 호환).
- 루트 **`서면/`** 폴더는 쓰지 않음(과거 복제본·이중 표기 폐지).

## 2. 이중 표기 — 증거(갑호증·법령정보)

| 표준 rel(문서·포털·다수 스크립트) | 구 트리(레거시, 일부 스크립트·실제 폴더) |
|-----------------------------------|------------------------------------------|
| `행정심판청구(증거)/최종/갑호증/` | `행정심판청구(증거)/갑호증/` |
| `행정심판청구(증거)/최종/법령정보/` | `행정심판청구(증거)/법령정보/` |

- **`web/commission-portal/public/app.js`** — `GAB_REPO_PATH_MARKER`, `serve`용 rel은 **`…/최종/…`** 기준.
- **`start.js`** — `…/최종/…` 요청 시 디스크에 없으면 **`…/증거/` 바로 아래** 동일 꼬리 경로로 fallback.
- **클론 예시**: `행정심판청구(증거)/최종/` 없이 **`갑호증`·`법령정보`** 가 증거 루트에 바로 있는 구성이 있을 수 있음.

## 3. 루트 폴더 역할(요약)

| 경로 | 용도 |
|------|------|
| `행정심판청구(최종)/` 직하 | **제출 정본 MD** `260404_01_…`·`260404_02_…`·별첨·시간축·집대성 등 (`260404_*`). DOCX 빌드 입력. |
| `행정심판청구(최종)/260404/` | **작업·허브** 개요·갑 검수·별지 드롭다운·판례주석 분리본 (`260404/260404_*`). 포털 개요 탭도 이 경로를 가리킴. |
| `행정심판청구(최종)/260403/260402/` | **2026-04-04 재작성 이전** 스냅샷(폴더명·파일명 접두 `260402`). 참고·복원용. |
| `docs/260403/260402/` | 당시 `docs/260402/` 와 동일 구조로 보관. (`docs/260403/` 루트에는 별지 DOCX 등 당일자 산출이 함께 있을 수 있음.) |
| `행정심판최종본/` | (선택) 예전 클론용 별칭 폴더. 포털은 **`행정심판청구(최종)/`** 가 없을 때만 동일 상대 경로로 읽기 시도. |
| `docs/260404/` | DOCX·제출용 txt·참고 추출 md **산출** — **Word 정본은 여기만**(행정심판청구(최종) 직하에 `.docx` 없음). |
| `docs/scripts/` | `260404_build_final_docx.py`, `export_submission_txt.py`, `extract_reference_section.py` |
| `행정심판청구(증거)/최종/` (권장) | 갑호증·법령정보·첨부 등 **제출 증거** 통합 루트. |
| `web/commission-portal/` | 심리 참고 포털. `public/data/portal-data.json`에 `tabSources`·`rel` 경로. |

## 5. 정리 시 권장

1. **새 작업**은 문서·포털과 같이 **`행정심판청구(증거)/최종/`** · **`행정심판청구(최종)/`** 를 기준으로 맞출 것.
2. **`행정심판최종본`** 을 쓰는 옛 클론이 있으면 **`행정심판청구(최종)`** 로 합치거나 junction으로 통일할 것.
3. **`260402_build_prec_pdfs.py`** 는 트리가 `최종` 없이 쓰이는 경우와 docstring이 어긋날 수 있음 → 실행 전 `ADJ_ROOT`·`LAW_INFO_DIR`·실제 폴더를 한번 확인할 것.
